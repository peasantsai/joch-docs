# Joch Docs

Documentation site for the Joch agent control-plane concept.

The source material came from `jochai.md` and has been split into a Material for MkDocs site covering architecture, specs, platform strategy, product naming, and repository planning. Kubernetes-style resource YAML specs live as one page per resource under `docs/specs/kubernetes/`, and platform strategy is split into focused pages under `docs/platform/`.
