# FAQ

## Is Joch AI an agent framework?

No. Joch AI does not compete with OpenAI Agents SDK, Claude Agent SDK, Google ADK, Microsoft Agent Framework, LangGraph, or CrewAI. It governs agents built with those systems.

## What is the MVP?

An open-source MCP/tool governance gateway and agent inventory system for teams running production AI agents across multiple frameworks.

## Why start with MCP and tools?

Every useful agent needs tools. Tool calls are where agents create side effects, touch sensitive data, and create audit requirements. MCP accelerates tool adoption and therefore increases the need for governance.

## What is an ABOM?

An Agent Bill of Materials. It lists the models, tools, MCP servers, memories, RAG sources, secrets referenced, policies, risk flags, and deployment targets associated with an agent.

## Does Joch AI require rewriting agents?

No. The MVP goal is to integrate through wrappers, callbacks, hooks, and proxy mode.

## Is Joch AI cloud-only?

No. The MVP must run locally, through Docker Compose, and on Kubernetes.
