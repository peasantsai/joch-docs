# License

The recommended project license is:

```text
Apache-2.0
```

Apache-2.0 is adoption-friendly for infrastructure software and aligns with the open-source core strategy.

## Commercial boundary

Commercial features can exist around:

```text
SSO/SAML
RBAC
advanced audit exports
policy packs
hosted dashboard
multi-tenant cloud
long-term trace retention
SIEM integrations
enterprise support
approval workflows at scale
compliance reporting
```

The core governance gateway, registry, policy engine, ABOM, Docker deployment, Helm chart, and framework adapters should remain open.
