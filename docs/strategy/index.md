# Strategy

Joch AI is positioned as an **agent control plane** with a beachhead in **MCP and tool governance**.

The product category:

```text
Agent Control Plane
MCP and Tool Governance Gateway
AI Agent Operations and Governance
Open-source control plane for agent fleets
```

## Core thesis

Teams will not adopt Joch AI because it is a better agent loop. They will adopt it because it secures and audits the tool layer across agents they already have.

## Core value proposition

> **Joch AI gives organizations one place to inventory, govern, audit, and operate AI agents across every framework, model, tool, and runtime.**

## Wedge

> **Joch Gateway: the secure MCP and tool-call governance layer for AI agents.**

The wedge is strong because every agent framework needs tools, MCP adoption creates tool sprawl, tool use creates security risk, and enterprises need one audit and policy layer.
