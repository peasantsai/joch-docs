# MVP v0.1

The MVP proves that Joch AI can sit in front of existing agent systems and deliver governance and visibility without forcing teams to rewrite their agents.

## MVP one-liner

> **Joch AI MVP is an open-source MCP/tool governance gateway and agent inventory system for teams running production AI agents across multiple frameworks.**

## MVP thesis

> Teams will adopt Joch AI if it can secure and audit their tool/MCP layer across existing agents.

## MVP must prove

```text
1. Joch can discover/register agents.
2. Joch can inventory their tools and MCP servers.
3. Joch can proxy and audit tool calls.
4. Joch can enforce policies before risky actions.
5. Joch can require human approval.
6. Joch can generate an Agent Bill of Materials.
7. Joch can run locally, in Docker, and on Kubernetes.
8. Joch can integrate without forcing teams to rewrite agents.
```

## MVP should not prove

```text
Joch is a better LangGraph.
Joch is a better CrewAI.
Joch is a better OpenAI Agents SDK.
Joch is a better Claude Agent SDK.
Joch has the best agent loop.
```
